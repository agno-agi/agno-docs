## How to use the template

1. Open the [template](https://agno.link/hack-repl)
2. Remix the template
3. Select any of the applications from the [Agno cookbooks](https://github.com/agno-agi/agno/tree/main/cookbook/examples/apps) and update the files accordingly.

## Testing in Replit

1. Click on Shell
2. Add your API keys
3. Run the command: `streamlit run app.py`

## Deploying your app

1. Click on Deployments
2. Add your API keys
3. Use the run command: `streamlit run app.py`
4. Deploy the app
